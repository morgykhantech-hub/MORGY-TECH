*2. Media Commands*
- `.sticker` - Turn image/video to sticker
- `.yt [link]` - Download YouTube video/song
- `.tiktok [link]` - Download TikTok
- `.ig [link]` - Download Instagram
- `.song [song name]` - Search and send song
- `.quote` - Send random bible/motivational quote

*3. Group Commands*
- `.menu` - Show all commands
- `.tagall` - Tag everyone in group
- `.welcome on/off` - Welcome message
- `.antilink on/off` - Delete links in group
- `.kick @user` - Remove member - admin only
- `.promote @user` - Make admin

*4. Fun & Tools*
- `.weather [city]` - Get weather
- `.bible [verse]` - Get bible verse
- `.joke` - Random joke
- `.pray` - Prayer message
- `.lyrics [song]` - Get song lyrics

*5. Owner/Admin Only*
- `.broadcast [message]` - Send to all users
- `.ban @user` - Ban user
- `.restart` - Restart bot
- `.ping` - Check bot speed
