*1. AI & Chat Commands*
- `.ai [question]` - Chat with ChatGPT
- `.gpt [question]` - Ask GPT-4
- `.gemini [question]` - Ask Google Gemini  
- `.claude [question]` - Ask Claude
- `.imagine [description]` - Generate AI image
- `.voice [text]` - Bot replies with voice note

- `.ytmp3 [link]` - Download YouTube audio only
- `.ytmp4 [link]` - Download YouTube video
- `.tiktok [link]` - Download TikTok no watermark
- `.ig [link]` - Download Instagram post/reel
- `.fb [link]` - Download Facebook video
